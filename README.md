# 🤖 AI Document Assistant

![Python](https://img.shields.io/badge/Python-3.9+-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-App-red)
![LangChain](https://img.shields.io/badge/LangChain-RAG-green)

An AI-powered document analysis assistant built using **Streamlit, LangChain, Groq, and ChromaDB**.
Upload a document and interact with it using AI to summarize, extract insights, generate quiz questions, or chat with the document.

---

## 🌐 Live Demo

🚀 [Open AI Document Assistant](https://your-streamlit-link.streamlit.app)
---

## 🚀 Features

* Upload **PDF or TXT documents**
* **Chat with your document**
* **Summarize documents**
* **Extract key insights**
* **Generate quiz questions**
* **Explain complex topics simply**

---

## 🛠 Tech Stack

* Python
* Streamlit
* LangChain
* Groq (Llama 3.1)
* HuggingFace Embeddings
* Chroma Vector Database

## ▶ Run Locally

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run streamlit_app.py
